**该项目的简洁说明，描述它的实现是什么：**<br>
1*实现互不干扰的两个灯* <br>
2*一个自己闪烁，一个通过按键来控制*<br>

**该项目所分配的引脚，要求使用表格书写，并为其描述该引脚所使用的功能：**<br>
1*使用的是野火MINI板 **“STM32F103RCT6”***  
2LED0通过 **PC2** 灯1  
LED1通过 **PC3** 灯2  
KEY0通过 **PA0** 按键0  

**该项目重要逻辑，要求使用代码块直接书写关键逻辑，并描述其作用:**  
1侧重于将彼此现象分割开来:定义last_led0_tick将上次执行完功能后的时间进行捕获，以便于下次通过**非堵塞延时**GetTick();进行延时  
2**current_button**用来读取当前按键状态
